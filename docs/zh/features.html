功能介紹

「終結內容農場」針對內容農場有三段阻擋功能：自動標示農場連結、阻止進入內容農場、封鎖廣告與惡意程式 ，且能自由訂閱網路黑名單或設定黑白灰名單。

三段阻擋內容農場

1. 自動標示農場連結

「終結內容農場」會自動把連往內容農場的網址加上標記，一眼就能看穿哪些超連結會連往內容農場。

有些網站的超連結會經過轉址處理，例如臉書會把連往 http://example.com 的網址轉寫為 https://l.facebook.com/l.php?u=https%3A%2F%2Fexample.com...。「終結內容農場」支援許多知名搜尋引擎、社群網站的轉址解析，此例若有內容農場位於 example.com，經前述轉寫的超連結也會被加上標記。目前支援的網站有 Google、Bing、DuckDuckGo、百度、搜狗、Facebook、Twitter、Instagram、YouTube 等等。

2. 阻止進入內容農場

「終結內容農場」會在使用者嘗試進入內容農場前自動攔截封鎖，以便及時回頭。

即使連往內容農場的超連結會被標記，仍可能有漏網之魚，使用者也可能因伺服端轉址、網頁腳本等非超連結方式重導向到內容農場，主動攔截功能可有效防止意外進入。

3. 封鎖廣告與惡意程式

少數情況可能需要查看被封鎖的頁面，此時可在封鎖頁面按下 [檢視]，「終結內容農場」會顯示去除廣告及程式碼的頁面，如此既能安全調查網頁內容，又不致讓農場主取得廣告流量及收益。

有些內容農場刻意設計成必須透過腳本載入頁面（可能是為了強迫使用者啟用網頁腳本，以利執行廣告或惡意程式），去除程式碼的 [檢視] 頁面無法正常運作。終結內容農場也提供了 [解鎖] 功能，可暫時解除對該網頁的封鎖，使用時須輸入驗證碼，且設有冷卻計時，以免頻繁解鎖導致失去封鎖意義。

如果調查後認為該網站不是需要抵制的內容農場，或有其他特殊需求，可點擊 [調整封鎖名單] 進入選項頁面把該網站從黑名單移除或加入灰、白名單。

自訂封鎖名單

「終結內容農場」可訂閱網路黑名單，訂閱後會定期取得最新版本的內容農場名單，以免遺漏層出不窮的新農場。

「終結內容農場」安裝後會自動訂閱一個預設的網路黑名單。如有額外需求，可在選項頁面的 [網路黑名單] 欄位加入網址訂閱更多黑名單；若預設黑名單不符需求，刪除其網址即可取消訂閱。

訂閱後可在選項頁面點擊 [檢視已載入的黑名單]，檢視目前已載入的黑名單及其最後更新時間。

除了預設的網路黑名單，「終結內容農場」也另外提供了幾份網路黑名單，可點擊 [我們的網路黑名單] 進入後台資料庫查看，並視需求加入訂閱。

「終結內容農場」的黑名單採開放舉報機制，可從選項頁面最後的 [舉報內容農場] 連結進入舉報表單舉報新發現的農場。舉報記錄、審核標準、審核結果都公開在網路上可供檢查及討論。

除了訂閱網路黑名單，也可以在選項頁面自訂額外的封鎖名單：

  • 黑名單：想標示及封鎖的網站。
  • 白名單：想排除的網站。列入白名單的網站即使被列入黑名單也不會被封鎖。
  • 灰名單：想排除的規則。列入灰名單的規則會使對應的黑名單規則失效。可用於排除網路黑名單中不想要的規則。

「終結內容農場」的封鎖名單格式相當簡單，只要輸入網域即可封鎖該網域及其所有子網域，例如 example.com 會封鎖 example.com、www.example.com、sub.example.com 等網站。「終結內容農場」有自動修正功能，一般只要把網址貼上，儲存後即會自動轉換成對應的網域。

指定網域時可使用萬用字元，例如用 example.blogspot.* 封鎖 example.blogspot.com、example.blogspot.tw、example.blogspot.com.tw 等網站。

如果需要封鎖網域下的特定頁面，可用正規表示式指定網址型態，例如 /^https?://(?:[\w-]+\.)*\.example\.com/malicious(?=[/?#]|$)/i。

每個規則後可加上空白，並在空白後輸入說明文字，以提供相關背景資訊，例如 example.com 範例網站。一個小技巧是在規則前面加上空白，可使該規則被視為說明文字而失效，日後只要移除空白即可使之重新生效。

在任意網頁點擊「終結內容農場」工具列按鈕（或在封鎖頁面點擊 [調整封鎖名單]）進入選項頁面時，會在頁首顯示網址、重導向、相關封鎖規則等資訊。若涉及重導向，可點擊連結查詢最近的網路請求，可用於追查某些多次重導向的垃圾網站。

用右鍵選單快速封鎖網站

除了手動輸入網址或規則，也可以透過右鍵選單快速將網站加入黑名單。在網頁空白處按右鍵可將該網站加入黑名單（有些瀏覽器如 Firefox 也支援在分頁上按右鍵），在超連結上按右鍵可將超連結指向的網站加入黑名單，也可以選取多個超連結按右鍵把它們指向的網站一次加入黑名單。

設定網址轉換規則以便精準封鎖

「終結內容農場」黑名單主要是針對網域，但有些內容農場只是網域下的一小部分，封鎖整個網域並不適當。比如用右鍵選單把超連結 https://www.youtube.com/channel/ContentFarmVideos 加入黑名單，或在自訂黑名單加入此網址時，預設會封鎖 youtube.com，但實際上我們往往只想封鎖該頻道，而不是整個 YouTube。

此時可以在 [網址轉換規則] 加入一條轉換規則：/^https?://www\.youtube\.com/channel/([\w-]+)(?=[/?#]|$)/ /^https?://www\.youtube\.com/channel/$1(?=[/?#]|$)/，這麼一來加入此網址時就會先經過轉換而變成封鎖 /^https?://www\.youtube\.com/channel/ContentFarmVideos(?=[/?#]|$)/。
